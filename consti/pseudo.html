Pseudocódigo

O pseudocódigo é uma forma textual de representar algoritmos,
funções ou outros processos, utilizando uma linguagem próxima da natural,
mas combinada com estruturas comuns da programação.
Segue regras rígidas de sintaxe, mas organiza o raciocínio lógico
de maneira clara e estruturada, facilitando a compreensão e o planeamento.

Diz-se pseudocódigo porque não é um código executável,
mas uma representação simplificada de um algoritmo ou programa.
Embora não haja uma sintaxe fixa, é comum usar comandos como:

* INÍCIO e FIM: Para assinalar o início e o fim do algoritmo.

* LEIA ou LER: Para receber dados do utilizador.

* ESCREVA ou ESCREVER: Para exibir informações.

* SE...ENTÃO...SENÃO: Para criar decisões lógicas.

* Declarações de variáveis: Como NOME_VARIAVEL: tipo para definir variáveis.
